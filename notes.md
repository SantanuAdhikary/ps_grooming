8. inline , block , inline block level element 

   element : combination of tags and content is called as element.
     eg: 
         <h1> good evening everyong </h1>

    inline level element : 
              this element content will display in same line.

              here we can't give height and width.

              eg:  <b>hi </b>
                   <i> bye </i>

     block level element 
           
            here content will display in different line.

            here we can give height and width.

        eg:
            <h1></h1> , <p></p>

     
     inline-block level element 
        
         it is combination of inline and block level element.
         here it will display in same line but we can provide height and width.

         eg:  <button></button> , <input> tag
